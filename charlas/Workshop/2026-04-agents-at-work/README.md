## De qué trata

Un workshop práctico donde construís tu propio agente de IA desde cero: con cerebro, herramientas, memoria y un canal de comunicación real. Al final tenés un agente corriendo en tu máquina, conectado a Telegram, respondiendo mensajes y ejecutando tareas de forma autónoma.

---

## Temario

### Conceptos
- **¿Qué es un agente?** — la diferencia entre un LLM y un agente que actúa
- **Anatomía de un agente** — LLM (cerebro), Tools (herramientas), Context (memoria), Interface (canal)
- **¿Qué puede hacer tu agente?** — reportes automáticos, respuestas proactivas, automatización de tareas

### OpenClaw
- **¿Qué es OpenClaw?** — plataforma open source construida sobre Claude Code
- **El Gateway** — cerebro central: cómo orquesta herramientas y contexto
- **SOUL.md y SKILL.md** — cómo definir la personalidad y capacidades del agente

### Instalación y configuración
- Setup del entorno local
- Conexión a Telegram
- Primer mensaje: verificar que el agente responde

### Personalización
- **SOUL.md** — identidad, tono y comportamiento del agente
- **Skills** — crear habilidades reutilizables: estructura, comunidad, ejemplos

### Seguridad
- Límites reales: qué puede y qué no puede hacer el agente
- Buenas prácticas para no exponer datos sensibles

---

## Requisitos

- Node.js 22+
- Cuenta de Telegram
- API key de OpenRouter
- VPS o máquina local *(no recomendado en tu máquina personal si no sabés lo que hacés)*
